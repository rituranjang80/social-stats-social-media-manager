

Requirements:
- Redesign only the UI/UX and layout. Do NOT remove or break any existing functionality, business logic, API, or database operations.
- Keep all existing features fully functional while adding new UI enhancements where appropriate.
- Build a modern, responsive, mobile-first interface with 100% vertical and horizontal responsiveness.
- Follow enterprise-level design with reusable, scalable architecture.

Frontend:
- React.js with fully generic, reusable, plug-and-play T-Type components.
- SCSS/SASS only (no inline CSS), using variables, mixins, functions, nesting, and modular architecture.
- Components must be reusable across different modules and projects.
- Clean folder structure, lazy loading, accessibility (WCAG), dark/light theme support, and optimized performance.

Backend:
- Django REST Framework.
- Create generic, reusable, plug-and-play REST APIs, serializers, services, permissions, validators, and utility classes.
- Follow SOLID principles, DRY, clean architecture, dependency injection where applicable, and proper error handling.
- Do not duplicate code; refactor into reusable modules.

General Standards:
- Production-ready, enterprise-grade code.
- Strong typing where applicable.
- Modular, maintainable, documented, and easily extendable.
- No hardcoded values; use configuration, constants, and environment variables.
- Backward compatible with the existing project.
